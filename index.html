Caveat Emptor:

These charts are made with data from data.nola.gov. Data are published on a 48 hour delay, so are never too up-to-date. Data also are not necessarily revised. So if, for instance, an incident initially published as a homicide is later determined to be an accident, then this change will not necessarily make it to data.nola.gov and this page's homicide count will be inaccurate. It is simply a limitation of the dataset that is available to us.

Additionally, and most imporantly, these data are only calls for service with a report to follow. Each datum is not necessarily an actual crime, only an instance of an NOPD unit dispatched to a scene that is interesting enough to warrant a report. Therefore, the data are only a rough approximation of actual reported crimes. Reports-to-follow typically mean the incident required some action on the part of the NOPD, and so typically do correspond to a crime, but not always.
